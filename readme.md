# PHP JWT AUTHENTICATION


## Guide

<br>

- Clone this repo :

  ```
    $ git clone https://github.com/antheiz/belajar-jwt-auth.git
  ```

- Open this project

  ```
    $ cd belajar-jwt-auth
  ```

- Install Library 

  ```
    $ composer require vlucas/phpdotenv
    $ composer require firebase/php-jwt
  ```

- Import SQL File [_learn_jwt.sql_]

- Run Project

  ```
    $ php -S localhost:8000
  ```
